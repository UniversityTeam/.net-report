Опис об'єкта управління

Об’єктом автоматизації є відео-хостинг. Користувач спочатку реєструється, тоді авторизується в системі. Після авторизації він може переглядати доступні у каталозі відеоматеріали, оцінювати так коментувати їх. Після кількох переглянутих відео стане доступним список рекомендацій, на основі історії переглядів. Також є можливість переглянути «випадкове» відео з доступних на сайті.

Розширений функціонал доступний для адміністраторів сервісу, а саме:

  • додавання та видалення відеоматеріалів (адміністратори та модератори);
  • модерація коментарів, їх підтвердження або видалення (адміністратори та модератори);
  • призначення нового адміністратора чи модератора (лише для адміністраторів);
  • блокування користувачів, за образи, рекламу чи з інших відомих причин (адміністратори та модератори);

Мета проекту

Метою створити зручний сервіс для перегляду відео, враховуючи власні побажання та досвід схожих сайтів.

  1. Максимально спростити доступ користувачів до відеоматеріалів;
  2. Налаштування швидкого фідбеку;
  3. Оптимізація процесу завантаження відео (скорочення часу між завантаженням та можливістю перегляду);
  4. Створення простого та зрозумілого інтерфейсу.

Бізнес задачі

  1. Реєстрація облікового запису та авторизація користувача.
  2. Перегляд фільму.
  3. Оцінка та коментування (необов’язково).
  4. Створення власного списку рекомендацій.

Бізнес процеси

  1. Створити обліковий запис та авторизувати користувача.
  2. Обрати фільм для перегляду.
  3. Оцінити фільм та написати коментар.
  4. Переглянути список рекомендацій.

Сценарії системи

  1. Незареєстрований користувач:
    • створити обліковий запис (зареєструватись).
    • пройти авторизацію (перегляд можливий і без авторизації).
    • вибрати фільму з каталогу для перегляду.
    • переглянути список рекомендацій (за бажанням).
  2. Зареєстрований користувач:
    • вибрати фільму з каталогу для перегляду.
    • оцінити та прокоментувати фільм (необов’язково).
    • переглянути список рекомендацій (за бажанням).
